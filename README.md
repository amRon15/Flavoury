# Recipe Sharing App

## Introduction
The Recipe Sharing App is a mobile application designed for the Android platform that allows users to upload, share, and discover delicious recipes. In addition to the recipe sharing functionality, the app also includes a basic social media feature, enabling users to interact with each other's creations.

## Key Features
- Recipe Sharing: Users can upload and share their own unique recipes, including ingredients, instructions.
- Social Interaction: Users can view, like the recipes shared by other users.
- Recipe Discovery: The app provides a searchable and browsable database of user-generated recipes, allowing users to discover new and inspiring culinary creations.
- User Profiles: Each user has a personalized profile where they can showcase their shared recipes and view their activity, such as number of creation and likes received.
- Recipe Bookmarking: Users can save their favorite recipes to a personal collection for easy access and reference.

## Technology
- Platform: Android
- Programming Language: Java
- Database: Firebase Firestore
- User Authentication: Firebase Authentication
- UI Framework: Android Studio with Material Design components
